介绍

  • writing-mode可以改变文字的显示方向，比如可以通过writing-mode让文字的显示变为垂直方向
  • 更重要的是它还会改变所有水平方向上的css属性
  • 比如text-align，通过writing-mode和text-align就可以做到水平和垂直方向的居中了

实现

  1. 父容器设置为垂直方向显示，并这是文本居中text-align
  2. 增加一个中间容器，设置水平方向显示(恢复为正确的方向)，并设置宽度为100%，以便让子元素居中
  3. 子元素现在相对于父元素垂直居中了，可以通过margin或行居中的方式实现水平居中

缺点

  1. 需要增加一个容器做转换
  2. 理解复杂
  3. 实现麻烦
