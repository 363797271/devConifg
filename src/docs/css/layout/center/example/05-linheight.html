介绍

  1. 将子元素设置为行内块元素
  2. 设置父元素的text-align，使行内块子元素水平居中
  3. 设置父元素的line-height和子元素的(line-height:initial;vertical-align: middle;)实现垂直居中
  4. 此时子元素的水平居中样式继承父元素，还要重置一下(text-align:left)

优点

  • 不需要知道子元素的宽高，也不需要固定宽高

缺点

  1. 宽度最大为容器宽度，高度超出容器，垂直方向置顶
  2. 需要父子都设置line-height
  3. 需要修改子元素的display
  4. 需要重置子元素的水平样式
